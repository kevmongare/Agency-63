# Agency 63 

A demo website for Agency 63 Agency 63 is a Kenyan creative agency specializing in below-the-line (BTL) marketing and experiential events.

## Project Structure


```agency-63/
├── src/
│   ├── components/
│   │   ├── Navbar.tsx        ← Sticky nav, mobile menu, CTA
│   │   ├── Hero.tsx          ← Full-screen hero, stats strip
│   │   ├── Clients.tsx       ← Scrolling marquee of brand logos
│   │   ├── Services.tsx      ← 6-service grid (BTL services)
│   │   ├── Work.tsx          ← Filterable case studies grid
│   │   ├── About.tsx         ← Image collage + brand philosophy
│   │   ├── Team.tsx          ← Team cards with grayscale hover
│   │   ├── Testimonials.tsx  ← 3-column client reviews
│   │   ├── Contact.tsx       ← Contact form + details
│   │   └── Footer.tsx        ← Multi-column footer
│   ├── App.tsx
│   ├── main.tsx
│   └── index.css
├── tailwind.config.js
└── package.json
```

To work on this project locally 

```js
npm install
npm run dev
```
